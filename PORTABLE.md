# Code Summarizer Portable

Run `Code Summarizer.exe` directly; no Node.js or Rust installation is required.

Ollama and at least one local model are separate requirements. The portable build also requires the Microsoft Edge WebView2 Runtime already present on the computer. The MSI and NSIS installers include Microsoft's offline WebView2 installer.

The app does not self-update. Verify `SHA256SUMS.txt` from the release before running the executable.
